# QUICK START GUIDE 🚀

## البدء السريع

### المتطلبات
- Python 3.8+
- MT5 مثبت على الجهاز
- اتصال بحساب تداول real أو demo

### التثبيت

```bash
# 1. استنساخ المستودع
git clone https://github.com/hmeda4696-hash/Trading-ML-System.git
cd Trading-ML-System

# 2. إنشاء بيئة افتراضية
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 3. تثبيت المكتبات
pip install -r requirements.txt
```

### التشغيل الأول

```bash
# تشغيل التطبيق
python src/main.py
```

### خطوات الاستخدام

#### 1. 📊 إضافة الأزواج
- انقر على تبويب "الأزواج"
- أدخل رمز الزوج (مثل EURUSD)
- اختر الإطار الزمني (M5, H1, D1)
- اضغط "إضافة الزوج"

#### 2. 🔍 تحليل الزوج
- انتقل إلى تبويب "التحليل"
- حدد فترة الاختبار (1-60 شهر)
- اختر نوع الاستراتيجية (تلقائي مرغوب)
- اضغط "بدء التحليل"

#### 3. 📈 مراجعة النتائج
- انتقل إلى تبويب "النتائج"
- شاهد الإحصائيات والإشارات
- ارجع الأرباح والخسائر
- معدل نجاح الاستراتيجية

#### 4. 💾 حفظ المؤشر
- اضغط "توليد المؤشر"
- سيتم حفظ المؤشر بصيغة MQL5
- انسخ المؤشر إلى مجلد MT5

### المميزات الرئيسية ⭐

✅ **تحليل ذكي**
- تحليل تلقائي للاتجاهات
- كشف مستويات الدعم والمقاومة
- نظام ML متقدم

✅ **Backtesting قوي**
- اختبار الاستراتيجيات التاريخية
- حساب دقيق للأرباح والخسائر
- رسم منحنى الأسهم (Equity Curve)

✅ **توليد مؤشرات MQL5**
- مؤشرات احترافية جاهزة للاستخدام
- دعم كامل لـ MT5
- حفظ تلقائي للملفات

✅ **واجهة رسومية احترافية**
- PyQt5 مع ثيم داكن جميل
- سهلة الاستخدام
- دعم اللغة العربية

### نصائح للاستخدام الأمثل 💡

1. **ابدأ بـ timeframes كبيرة** (H1, D1) للنتائج الأفضل
2. **استخدم فترات اختبار طويلة** (6-12 شهر) للدقة
3. **راجع النتائج قبل التداول الفعلي**
4. **حدّث البيانات بانتظام**
5. **استخدم stop loss و take profit دائماً**

### استكشاف الأخطاء 🔧

#### MT5 غير متصل
```
✗ Error: Failed to initialize MT5
✓ الحل: تأكد من تثبيت MT5 وأنه مفتوح
```

#### لا توجد بيانات
```
✗ Error: No data for SYMBOL
✓ الحل: تحقق من رمز الزوج وأن الزوج متاح في حسابك
```

#### مكتبات ناقصة
```
✗ ModuleNotFoundError
✓ الحل: أعد تشغيل: pip install -r requirements.txt
```
