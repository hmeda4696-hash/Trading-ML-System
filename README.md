# Trading ML System 🤖📊

نظام تحليل وتعلم آلي متقدم للأزواج الفوركس مع واجهة رسومية احترافية PyQt5

## المميزات ✨

- 🔍 **تحليل ذكي للأزواج** - اكتشاف تلقائي للستراتيجية المناسبة
- 🤖 **نظام تعلم آلي** - ML-based analysis
- 📈 **Backtesting قوي** - اختبار الاستراتيجيات التاريخية
- 🎨 **واجهة PyQt5 احترافية** - سهلة الاستخدام
- 💾 **حفظ المؤشرات** - حفظ تلقائي بصيغة MQL5
- 📊 **عرض النتائج** - إحصائيات وتقارير مفصلة

## التثبيت 🚀

```bash
# 1. استنساخ المستودع
git clone https://github.com/hmeda4696-hash/Trading-ML-System.git
cd Trading-ML-System

# 2. إنشاء بيئة افتراضية
python -m venv venv
source venv/bin/activate  # على Windows: venv\Scripts\activate

# 3. تثبيت المكتبات
pip install -r requirements.txt

# 4. تشغيل التطبيق
python src/main.py
```

## الاستخدام 💡

### خطوات تشغيل سريعة:

1. **إضافة الزوج**
   - اكتب رمز الزوج (مثل: EURUSD, GBPUSD)
   - حدد الـ Timeframe (M5, M15, H1, إلخ)

2. **التحليل التلقائي**
   - انقر على زر "تحليل"
   - يقوم النظام بـ:
     - جلب البيانات من MT5
     - تحليل الاتجاهات
     - تحديد الستراتيجية المناسبة

3. **الاختبار التاريخي**
   - حدد فترة الاختبار
   - شاهد معدل النجاح والأرباح

4. **حفظ المؤشرات**
   - انقر "حفظ المؤشر"
   - يتم حفظه تلقائياً بصيغة MQL5
   - جاهز للاستخدام في MT5

## المكتبات المطلوبة 📦

- **PyQt5** - الواجهة الرسومية
- **pandas** - معالجة البيانات
- **numpy** - الحسابات العددية
- **scikit-learn** - التعلم الآلي
- **ta-lib** - المؤشرات الفنية
- **MetaTrader5** - الاتصال مع MT5
- **matplotlib** - رسوم بيانية

## الستراتيجيات المدعومة 🎯

- ✅ Trend Following (تتبع الاتجاه)
- ✅ Mean Reversion (العودة للمتوسط)
- ✅ Breakout (كسر المستويات)
- ✅ Support/Resistance (الدعم والمقاومة)
- ✅ Multi-timeframe Analysis

- Custom ML-based Indicators
